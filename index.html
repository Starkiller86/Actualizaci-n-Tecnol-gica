Introducción a html

¿Qué es HTML?

  • Estándar para Lenguaje de Híper Texto de maquetado o Hyper Text Markup Language
  • Describe la estructura de las páginas web
  • Consiste en elementos
  • Estos elementos le dicen al navegador como mostrar el contenido
  • Los elementos son etiquetas con contenido

Elementos básicos explicados

  1. La etiqueta "DOCTYPE" declara el documento como HTML
  2. La etiqueta "html" es el elemento principal de una página HTML
  3. La etiqueta "head" contiene información adicional acerca de la página HTML
  4. La etiqueta "title" especifica el título de la pagina HTML
  5. La etiqueta "body" especifica el cuerpo de la página web (todos los elementos visibles)
  6. La etiqueta "h1" define el texto como un título
  7. La etiqueta "h2" define el texto como un subtítulo
  8. La etiqueta "p" define el tamaño del texto como párrafo

¿Qué es un elemento HTML?

Un elemento HTML está definido por una etiqueta de inicio, seguido de algún contenido y una etiqueta de finalización.
  • <nombreEtiqueta>: etiqueta de inicio
  • Contenido: entre la etiqueta de inicio y la etiqueta de finalización
  • </nombreEtiquta>: etiqueta de finalización

Atributos HTML

Los Atributos HTML proporcionan información adicional acerca de los elementos HTML (siempre son especificados en la etiqueta de inicio) & lt ; se utiliza para menor que, & gt ; se utiliza para mayor que.

Elementos de Formato

  • <b>Un texto en negritas</b>: texto en negritas
  • <strong>Un texto en negritas</strong>: texto en negritas
  • <i>Texto italico</i>: Texto italico
  • <mark>Texto resaltado</mark>: Texto resaltado

Etiqueta <map>

La etiquta <map> define el mapa de una imagen. Una imagen mapeada es una imagen que tiene distintas areas detectables por el navegador,
logrando de manera independiente cada una de las areas. La etiqueta <map> define las areas de la imagen mediante la etiqueta <area> con los atributos HTML "shape",
que define la forma del area señalada en la imagen, "coords" que define las coordenadas de cada uno de los puntos e la shape, "href"
que puede definir un hipervínclulo para cada una de las areas

Ejemplo:

<img src="workflow.jpg"usemap="#workmap">
<map name="workmap">
<area shape ="rect" coord="34,44,270,350" href="pagina1.com">
<area shape ="circle" coord ="200,172,333,250" href = "pagina2.com">
</map>

Elemento Table en HTML

La etiquta <table> permite colocar los datos en filas y columnas usando sus derivaciones <tr> para definir una fila (table row),
<th> para definir los encabezados de la tabla (table header) y <td> para asignar valores a las filas correspondientes a los encabezados de
las columnas (table data).

Ejemplo:

<table>

<tr>
<th>Nombre</th>
<th>Apellido</th>
<th>Edad</th>
<th>Dirección</th>
</tr>

<tr>
<td>Eduardo</td>
<td>Campos</td>
<td>18</td>
<td>Calle 28 #3 Col. Arboledas Querétaro</td>
</tr>

<tr>
<td>Ashley</td>
<td>Morales</td>
<td>22</td>
<td>Calle Rios Norte #12A Viñedos Querétaro</td>
</table>
